### The AI Native Tiny Little Website System

Current executable size: **30MB**

### Features

⭐️ **Single executable**

⭐ **No prerequisites**

**Content**
- Markdown based content management
- Asset management (images, videos, other files)
- Video streaming
- Role based access management
- Tag based content organisation
 
**Tech**
- Web and command line live administration and configuration
- TLS: BYO manual certs, self signed and ACME (HTTP, DNS)

**AI**
- Work in progress
- Initial focus on administration

### Usage

If you trust the builds, download the latest release for your operating system and run it in an empty directory.
NoPressure will set itself up and display the admin password. Please change it immediately.

If you don't trust the builds, please refer to [build and run](BUILD.md) on how to build from source.

Check out the [CLI](CLI.md) documentation for administering the system from the command line.
